La verificación de la API de texto de lienzo utiliza la técnica de detección #2 . Si su navegador admite la API de lienzo , el objeto DOM que crea para representar
un canvas elemento tendrá el getContext()método . Si su navegador no admite la API de lienzo , el objeto DOM que crea para un canvas elemento solo tendrá el
conjunto de propiedades comunes, pero nada específico del lienzo
