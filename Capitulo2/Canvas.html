HTML5 define el elemento 'canvas' como "un lienzo de mapa de bits dependiente de la resolución que se puede usar para representar
gráficos, gráficos de juegos u otras imágenes visuales sobre la marcha". Un lienzo es un rectángulo en su página donde puede usar JavaScript
para dibujar lo que quiera. HTML5 define un conjunto de funciones ("la API de lienzo ") para dibujar formas, definir rutas, crear degradados y aplicar transformaciones.

La verificación de la API de lienzo utiliza la técnica de detección #2 . Si su navegador admite la API de lienzo , el objeto DOM que crea para representar un
canvas elemento tendrá un getContext()método . Si su navegador no admite la API de lienzo , el objeto DOM que crea para un canvas elemento solo tendrá el conjunto
de propiedades comunes, pero nada específico del lienzo.
