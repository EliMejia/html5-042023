HTML5 define el elemento como "un lienzo de mapa de bits dependiente de la resolución que se puede usar para representar
gráficos, gráficos de juegos u otras imágenes visuales sobre la marcha". Un lienzo es un rectángulo en su página donde puede usar JavaScript
para dibujar lo que quiera. HTML5 define un conjunto de funciones ("la API de lienzo ") para dibujar formas, definir rutas, crear degradados y aplicar transformaciones.
