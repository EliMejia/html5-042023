Web Workers proporciona una forma estándar para que los navegadores ejecuten JavaScript en segundo plano. Con los trabajadores web, puede generar
varios "hilos" que se ejecutan al mismo tiempo, más o menos.
La comprobación de trabajadores web utiliza la técnica de detección n.º 1 . Si su navegador es compatible con Web Worker API , habrá una Workerpropiedad
en el objeto global window. Si su navegador no es compatible con Web Worker API , la Workerpropiedad no estará definida.
