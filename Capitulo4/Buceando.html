Buceando

TML 5 define el elemento canvas como "un lienzo de mapa de bits dependiente de la resolución que se puede usar
para representar gráficos, gráficos de juegos u otras imágenes visuales sobre la marcha". Un lienzo es un
rectángulo en su página donde puede usar JavaScript para dibujar lo que quiera
Un canvas elemento no tiene contenido ni borde propio.
Se puede tener más de un canvas elemento en la misma página. Cada lienzo se mostrará en el DOM y cada lienzo
mantiene su propio estado. Si asigna un id atributo a cada lienzo, puede acceder a ellos como a cualquier otro elemento.

Para poder encontrar fácilmente ese canvas elemento en el DOM .

var a_canvas = document.getElementById("a");
