CAMPOS DE ENFOQUE AUTOMÁTICO

Los sitios web pueden usar JavaScript para enfocar el primer campo de entrada de un formulario web automáticamente.
Debido a que el enfoque automático se realiza con JavaScript, puede ser complicado manejar todos estos casos extremos, y hay pocos
recursos para las personas que no desean que una página web "robe" el enfoque.

Para resolver este problema, HTML5 introduce un autofocusatributo en todos los controles de formulario web . El autofocusatributo hace
exactamente lo que dice en la lata: tan pronto como se carga la página, mueve el foco de entrada a un campo de entrada en particular.
Pero debido a que es solo marcado en lugar de script, el comportamiento será consistente en todos los sitios web. Además,
los proveedores de navegadores (o autores de extensiones) pueden ofrecer a los usuarios una forma de desactivar el comportamiento de enfoque automático.

Los navegadores que no admitan el autofocusatributo simplemente lo ignorarán.
