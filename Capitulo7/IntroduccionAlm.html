INTRODUCCIÓN AL ALMACENAMIENTO HTML5

El almacenamiento HTML5 es una forma de que las páginas web almacenen localmente pares clave/valor con nombre, dentro del navegador
web del cliente. Al igual que las cookies, estos datos persisten incluso después de navegar fuera del sitio web, cerrar la pestaña
del navegador, salir del navegador o lo que sea. A diferencia de las cookies, estos datos nunca se transmiten al servidor web
remoto. A diferencia de todos los intentos anteriores de proporcionar almacenamiento local persistente, se implementa de forma
nativa en los navegadores web, por lo que está disponible incluso cuando los complementos de navegador de terceros no lo están.

¿Qué navegadores?

Bueno, la última versión de casi todos los navegadores admite almacenamiento HTML5 ... ¡incluso Internet Explorer!
