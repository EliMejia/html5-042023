VISIONES EN COMPETENCIA

Una visión es un acrónimo que probablemente ya conozcas: SQL . En 2007, Google lanzó Gears , un complemento de navegador cruzado de
código abierto que incluía una base de datos integrada basada en SQLite . Este primer prototipo influyó más tarde en la creación de
la especificación Web SQL Database . Web SQL Database (anteriormente conocido como "WebDB") proporciona un envoltorio delgado
alrededor de una base de datos SQL

Esta cadena puede ser cualquier declaración de SQL admitida , incluidas las declaraciones SELECT, UPDATE, INSERTy DELETE. Es como la
programación de una base de datos back-end,
Todos los implementadores interesados ​​han utilizado el mismo backend de SQL (Sqlite), pero necesitamos múltiples implementaciones
independientes para proceder a lo largo de una ruta de estandarización. Hasta que otro implementador esté interesado en implementar
esta especificación, la descripción del dialecto SQL se ha dejado como una simple referencia a Sqlite, que no es aceptable para unestándar. Es en este contexto que les presentaré otra visión de la competencia para el almacenamiento local persistente y avanzado para
aplicaciones web: la API de base de datos indexada , anteriormente conocida como "WebSimpleDB", ahora conocida cariñosamente como "IndexedDB".
