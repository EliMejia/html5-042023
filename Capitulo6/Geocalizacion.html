Gelocalización

La geolocalización es el arte de averiguar dónde se encuentra en el mundo y (opcionalmente) compartir esa información con personas de confianza.
Hay más de una forma de averiguar dónde se encuentra: su dirección IP , su conexión de red inalámbrica, con qué torre
de telefonía celular está hablando su teléfono o un hardware de GPS dedicado que calcula la latitud y la longitud a partir de la
información enviada por los satélites en el cielo.

La API de Gelocalización


La API de geolocalización le permite compartir su ubicación con sitios web de confianza. La latitud y la longitud están disponibles
para JavaScript en la página, que a su vez puede enviarla de regreso al servidor web remoto y hacer cosas sofisticadas para conocer
la ubicación, como encontrar negocios locales o mostrar su ubicación en un mapa.

Junto con la compatibilidad con la API de geolocalización estándar , hay una gran cantidad de API específicas para dispositivos en
otras plataformas móviles.
